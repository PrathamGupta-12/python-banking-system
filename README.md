# python-banking-system
A console-based Python banking system featuring account creation, deposits, withdrawals, balance inquiry, and transaction management.

## Features
- Create an account
- Deposit money
- Withdraw money
- Check account balance
- Display all account
- Check balance
- Transfer money
- Transaction History

## Technologies
- Python 3

## Note
This is one of my early Python projects. The focus was on implementing the functionality, so the code structure is not yet optimized. I plan to refactor and improve it as I continue learning object-oriented programming and software design.
